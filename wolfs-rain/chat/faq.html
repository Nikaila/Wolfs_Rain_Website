よくある質問と答え

チャットの画面が表示されません

お使いの環境に、最新版の Flash Player がインストールされているかどうかをご確認ください。 最新版の Flash Player は、マクロメディア社のページからダウンロードできます。
チャットのページを表示しようとしたら、ダイアログが表示されました。これは何ですか？
Windows 版 Internet Explorer でご覧になられた場合、お使いの環境の Flash Player が古い場合は自動アップデートの機能が起動して、プラグインが最新版に入れ替わります。ダイアログはそれをお知らせするものです。
「CONNECT ERR」「接続失敗」というメッセージが表示されます
サーバーに接続できなかった場合に、これらのメッセージが表示されます。このチャットではリアルタイム通信のために特殊なポートを使用しますので、ファイアーウォールがある環境ではご利用になれない場合があります。
ファイアーウォールとは？

外部からの不正な侵入を防ぐために用意されている機能のことです。ファイアーウォールでは、様々な通信用のポートを個別に管理／制御して、不必要な通信は除外するようになっています。この機能が有効になっている場合、本チャットはご利用になれない場合があります。

ファイアーウォールは多くの企業や CATV 等の一部のプロバイダで採用されているほか、ブロードバンドルーターや家庭用の OS （WIndows XP や MacOS X 等）にも含まれている場合があります。
ファイアーウォールによる制限かどうかを確認されたい場合は、プロバイダまたは企業のネットワーク管理者にお問い合わせ下さい。

なお、本チャットはネットワークゲームのように特殊なポートを使用して通信を行なっておりますので、ネットワークゲームがご利用になれない環境では、本チャットもご利用になれない可能性があります。

ファイアーウォールという名前は、火事の際に延焼を防ぐために用いられる「防火壁（firewall）」が由来です。

「DISCONNECT」「切断」というメッセージが表示されます。
「切断エラー」というダイアログが表示されます。

メンテナンスなどの何らかの理由でサーバーから切断された、または回線が通信不能状態になった等の場合に、サーバーとの接続が切れた状態を表します。この場合は、再接続を行なってください。
入室するとログが真っ白です
仕様です。サーバーにはログは記録していませんので、入室時に過去ログは表示されません。同じ部屋の中にいる人とメッセージを交換した場合だけ、その内容が画面上に過去ログとして表示されます。すでに会話中の部屋に入室した場合も、それまでの会話は見ることは出来ません。
入室したのですが、誰もいません。この場合、会話は出来ないのですか？
誰かがいないと、会話は出来ません。誰かが来るのを待ちましょう。このチャットはリアルタイムチャットですので、誰かが来たことはリアルタイムに分かります。
チャットルームが複数あるのはなぜですか？
既に行われているチャットとは違う話題で話をしたい場合などに利用できるように、3 つのチャットルームを用意いたしました。
チャットルームには人数制限はありますか？
ありません。
チャットルームには時間制限はありますか？
ありません。
チャットで使える文字に制限はありますか？
UTF-8 に対応しておりますので、多国語の入力／表示が可能です。
チャットでは HTML のタグは使えますか？
使えません。
メッセージが時系列に並んでいないことがあるのですが？
仕様です。
このチャットのプログラムはどこかで配付されていないのでしょうか？

「WOLF'S RAIN OFFICIAL CHAT」は、WOLF'S RAIN 公式ページ専用に開発されたチャットです。
本チャットのプログラム等の配付は行なっておりません。
